# Compass

**mod**: AdvancedPeripherals  
**peripheral type**: `compass`  
**source**: `CompassPeripheral.java`

## 概要

コンパス。タートル専用ペリフェラル。タートルが向いている方向を取得する。

## メソッド一覧

| メソッド名 | 引数 | 戻り値 | imm | 説明 |
|---|---|---|---|---|
| `getFacing` | — | `string` | ✗ | タートルの向いている方向を返す (`"north"`, `"south"`, `"east"`, `"west"`) |

## 備考

- このペリフェラルはタートル専用。通常コンピューターには接続できない。
